IntellJ中开发的第一个Jersey+Tomcat

  • Java

打算用Jersey来开发一个轻量级的RESTful web app。

借用网上这个例子来学习IntelliJ上使用Tomcat和Jersey：

http://blog.csdn.net/felcon/article/details/51832146

准备工作

  • IntelliJ IDEA：下载Ultimate版本。Windows版本直接点击安装。
  • Tomcat：下载Tomcat 8.0 ，zip包解压。
  • Maven：下载Maven，zip包解压，配置环境变量。

创建工程

例子是从建立了一个Restful web service开始的，然后添加的Web app服务和Maven。Maven中加上Jersey依赖这样来搭建环境。因为Maven会下载所有Jersey需要的lib，事先创建Restful web service而引入Jersey类有点多此一举。因此我是从建立Web Application开始的。
