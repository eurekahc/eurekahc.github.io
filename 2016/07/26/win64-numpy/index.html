Win64安装Numpy

  • 机器学习

1. 下载whl安装包

http://www.lfd.uci.edu/~gohlke/pythonlibs/#numpy

2. 安装whl

pip install SOME.whl

假设Python安装路径：C:\Python27
pip.exe路径：C:\Python27\Scripts\

如果pip版本低，先升级pip即可
python -m pip install –upgrade pip
